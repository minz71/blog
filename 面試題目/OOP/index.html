# call by value (傳值)

  • 在 Call-by-Value 中，函式的參數是被傳遞值的副本。
  • 在函式內部，對參數的修改不會影響到原始的變數。

# call by address (傳位置)

  • 傳了實際的記憶體位置進去 function
  • 也是 call by value 的變形，改為傳記憶體位址

# call by reference (傳參考)

  • 在 Call-by-Reference 中，函式的參數接受原始變數的參考（或記憶體位置）。
  • 在函式內部，對參數的修改會直接影響到原始的變數。
