# API 使用範例 - 氣象資料開放平台

# 介紹

這篇文章介紹關於氣象資料開放平台 API 的註冊和使用說明，並提供了程式碼範例。

  1. 到氣象資料開放平台註冊帳號

  2. 註冊一個帳號 https://pweb.cwa.gov.tw/emember/register

  3. 到電子信箱啟用會員後再次登入

  4. 登入後取得授權碼並將授權碼複製下來

  5. 前往開發指南查看 api 文件

  6. 可以在 swagger 文檔中使用 API request

  7. 填入需要的參數後點擊 Execute 執行

  8. 執行後的回傳結果
