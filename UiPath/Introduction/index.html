# generative validation 產生驗證

  • 因為人工智慧模型可能出錯
  • 生成人工智慧作為第二意見去比對
  • 如果生成模型和專用模型最終都得到相同的結果，則跳過人工檢查

# UiPath Clipboard AI™

  • 使用圖片文字辨識
  • 自動填入表格等位置

# UiPath Document Understanding

  1. 不同的模板
  2. 手寫文件
  3. 簽名文件
  4. 勾選文件
  5. 不整齊或被旋轉的掃描文件
  6. 多樣的檔案格式
  7. 低品質的掃描文件

# UiPath Communications Mining

從發送電子郵件到回覆客戶請求。
在任何通訊管道上即時轉換為可操作的資料。
例如：辨識飯店名稱、房間類型和相關日期等資料，並將其輸入到自動化系統中。
