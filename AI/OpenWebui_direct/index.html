在 Open-webui 使用自己的 API key

Open-webui 在 0.5.11 版本後，使用者可以使用自己的 API key。

  1. 註冊一個帳號

如果出現帳號待啟用，需要通知管理員開啟權限

  1. 點選右上角的使用者的圖像，選擇設定

  1. 輸入 API URL 跟 key

  1. 回到首頁按下 F5 重新整理後刷新則會自動取得模型
