# 介紹

  • Hexo 是一個部落格框架，可以生成靜態網頁。

  • Shoka

是一個 Hexo 主題，Shoka 取的名字是書架，是一個為了閱讀筆記而生的主題，
相關介紹可以參考作者網站：https://shoka.lostyu.me/computer-science/note/theme-shoka-doc/

優萌初華

Shoka作者


Shoka 作者於部落格底下的留言中的回覆，
該項目可能不再更新，Github 上最後的 Commit 日期是2021/10/03

  • ShokaX

https://github.com/theme-shoka-x/hexo-theme-shokaX
ShokaX 是 Shoka 的精神續作，
原先 Shoka 使用了 JavaScript + Native + Nunjucks 的技術，
而 ShokaX 則使用了 TypeScript + Vue 3 + Pug 的技術搭配，
並新增了一些功能。

# 安裝

shokaX-CLI 已棄用，詳情: https://github.com/theme-shoka-x/shokaX-CLI

安裝文檔: https://docs-hexo.shokax.top/getting-started/

  1. 使用 shokaX can
    https://github.com/theme-shoka-x/shokax-can
git clone https://github.com/theme-shoka-x/shokax-can --depth=1
cd shokax-can
pnpm install

  1. 依說明文檔進行配置
    https://docs.kaitaku.xyz/guide/config.html

  2. 啟動 hexo server

hexo s # 如果報錯更換為 pnpm dlx hexo s

# 撰寫第一篇文章

  1. 在source/_posts資料夾下新增一個md檔案，例如：first.md。
  2. 在檔案內容中新增一些內容，例如：
title: 第一篇文章
date: 2024-04-24
tags: ShokaX
---
# 第一篇文章
這是內容
  1. 可以看到第一篇文章了。
