# 介紹

VMware Workstation 是由 VMware 開發的桌面虛擬機軟體，
博通（Broadcom）於 2023 年 11 月收購 VMware 後，
於 2024 年 5 月 14 日，宣布將以下虛擬化軟體供個人免費使用。

  • VMware Workstation Pro : 支援 Windows 與 Linux
  • VMware Fusion Pro : 支援 macOS

現在提供兩種授權模式：

  • 免費個人使用（Free Personal Use）
  • 付費商業使用（Paid Commercial Use）

# 下載 VMware

# 直接在檔案 server 下載

下載連結: https://softwareupdate.vmware.com/cds/vmw-desktop/ws/

選擇版本 (17.5.2) > 對應的資料夾 > 選擇系統 (linux/windows) > core > VMware-workstation-17.5.2-23775571.exe.tar

這個是主程式的安裝，tools 的部分到 packages 目錄下載。

選擇版本 (17.5.2) > 對應的資料夾 > 選擇系統 (linux/windows) > packages

下載連結: https://softwareupdate.vmware.com/cds/vmw-desktop/fusion/

選擇版本 (13.5.2) > 對應的資料夾 > universal > core > com.vmware.fusion.zip.tar

# Broadcom 官網

  1. 註冊登入 Broadcom 帳號

https://profile.broadcom.com/web/registration

  1. 下載 VMware

  • VMware Workstation Pro

  • VMware Fusion Pro

以下使用 windows 安裝目前的最新版本 (17.5.2)

  1. 點擊下載連結後選擇版本

  2. 同意條款後，點擊下載

  3. 填寫基本資料，點選 submit 送出

  4. 回到步驟 2 的頁面，點擊下載
